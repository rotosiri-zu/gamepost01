# 【内容】
- ゲームソフト紹介サイトをクリックするとホーム画面に戻る
- ログインIDPW/ログアウト機能
- 投稿機能
- ゲームソフト詳細画面
- 編集画面、編集機能
- 削除完了画面、削除機能
- ページネーション機能
- ransackによる検索機能 
- カンタンログイン機能
- RSpec
 ・統合テスト(system spec)
 ・機能テスト(request spec)
 ・単体テスト(モデル等)
- ハンバーガーメニュー機能

- カンタンログインでテストユーザーでログインできます
